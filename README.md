# upload_file
Server to upload files

# requirements - Libs 
python 2:
pip install requests  
pip install flask  
pip install -U Werkzeug  

python 3:
pip install requests  
pip install flask  
pip install -U Werkzeug 


Download Libs from https://pypi.org


**Instalation from local:**

1. Created folder:  
mkdir libs_upload  
cd libs_upload  

2. Download libs in folder:  
Flask,
Werkzeug,  
Jinja2,  
itsdangerous,  
click,  

3. Install libs:  
pip install *.whl
